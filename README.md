# 露天市集多商品共同賣家查詢器 (Ruten Multi-Item Finder)

本工具旨在解決使用者在露天市集採購多樣商品時，需手動比對各賣家與篩選條件不足的痛點。透過自動化腳本邏輯，精準定位同時具備多項目標商品的「共同賣家」，優化採購效率與運費成本。

---

## 核心功能
* 跨商品交叉比對：支援 N 筆商品搜尋，自動篩選出同時擁有所有目標物之賣家。
* 智慧過濾系統：提供「排除字段」功能，可自動過濾無關廣告或不符需求之商品資訊。
* 隱形執行模式：支援瀏覽器視窗移置螢幕外執行，確保在背景運行時不干擾使用者操作。
* 精準預算控制：可針對個別商品設定庫存下限與預算上限，確保結果符合採購需求。

---

## 技術規格
- 執行環境：Windows 10 / 11 (64-bit)
- 必要組件：需安裝 Google Chrome 瀏覽器
- 核心技術：Python Selenium 封裝

## 使用指南

### 1. 環境準備
* 本程式已封裝為執行檔，無需安裝 Python 環境。
* 請確保您的電腦已安裝 Google Chrome 瀏覽器。

### 2. 設定搜尋條件
* 包含字段：輸入您要尋找的商品關鍵字。
* 排除字段：輸入不想看到的關鍵字（如：二手、代購、已售完）。
* 庫存與預算：設定該商品至少需有多少庫存，以及單件最高預算。
* 等待秒數：建議設定為預設，更改視網路穩定度與網站載入速度而定。

### 3. 執行查詢
* 點擊「查詢」按鈕。
* 程式會先鎖定第一項商品的賣家名單，隨後依序進入各賣場確認是否具備後續之目標商品。
* 若不希望瀏覽器擋住畫面，請勾選「爬蟲時是否將模擬視窗移置螢幕外」。

### 4. 重要注意事項與操作建議
**程式無回應說明**：爬蟲執行期間，主視窗可能會出現「無回應」或無法拖動的狀態，此為程式正集中資源進行網頁解析的正常現象，請勿強行關閉，待搜尋完成後即可恢復正常。
* **請勿縮小化或關閉爬蟲視窗**：執行過程中，請勿手動將新開啟的 Chrome 瀏覽器視窗「縮小化」。由於作業系統與瀏覽器的節能機制，縮小化視窗會導致程序被自動卸載，進而造成搜尋程序中斷。
* **搜尋排序建議**：搜尋邏輯會以「第一列商品」的結果作為初始篩選基礎。建議將名稱最精確或品項較稀有的商品排在第一筆，能有效減少無效賣場的比對時間，顯著提高整體搜尋速度。
